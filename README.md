# 💰 Financial Dashboard Analysis - Power BI

## 📌 Overview

This project presents an **interactive financial dashboard** built using **Power BI** to analyze and compare **Actual vs Budget performance** across departments, accounts, and time.

The dashboard helps organizations monitor financial health, detect overspending, and make data-driven decisions.

---

## 🎯 Objectives

* Compare actual spending vs budget
* Identify departments exceeding budget
* Analyze financial trends over time
* Evaluate account-level performance
* Provide actionable financial insights

---

## 📊 Dataset Description

The dataset includes:

* Account Type
* Organization Name
* Department Group
* Actual Amount
* Budget Amount
* Variance
* Variance %
* Date (Year, Month)

---

## 📈 Key KPIs

* 💰 **Total Actual Amount:** 1.32bn
* 📊 **Total Budget Amount:** 42.41M
* 📉 **Variance %:** 30.03%
